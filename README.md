## Data Persistance
Sqlite is used to persist data in this solution. It is saved in the folder BookAPI with a name of 'bookcatalog.db'.
Entity Frameworkcore is used to interact with the Sqlite database.

## Launching Project
Ensure that the solution launches both the BookCatalogUI project and the BookAPI project at the same time. This will allow the UI to make its calls to the  Web API.
